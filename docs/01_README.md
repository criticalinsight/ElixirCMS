# Publii-Ex

Publii-Ex is a next-generation Desktop CMS built on the **Localhost Monolith** pattern. It provides a powerful, standalone environment for managing static websites with a modern UI and production-ready publishing features.

## Key Features
- **Integrated CMS:** Manage posts, static pages, and media from a single Elixir-powered dashboard.
- **Theming Engine:** Flexible filesystem-based themes using EEx templates.
- **Intelligent Search:** Built-in Pagefind integration for lightning-fast client-side search.
- **Professional SEO:** Full control over metadata, OpenGraph tags, RSS feeds, and sitemaps.
- **One-Click Deploy:** Publish directly to GitHub Pages.
- **Desktop Ready:** Package as a standalone `.exe` for Windows via Burrito.

## Quick Start
1. Install dependencies: `mix deps.get`
2. Build assets: `mix assets.deploy`
3. Run the monolith: `mix phx.server`
4. Access the CMS: `http://localhost:4000`
